# TdeP integración AWS

### Para el correcto funcionamiento de la integración favor considerar los siguientes pasos

- git pull del repositorio
- instalar node y npm
- node install
- npm install -g node-windows
- npm link node-windows
- crear un archivo llamado .env donde deberá estar la información de AWS
- node create_windows_services.js

  BUCKET_NAME = XXX
  IAM_USER_KEY = XXX
  IAM_USER_SECRET = XXX
